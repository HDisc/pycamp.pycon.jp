## How to build

```
$ clone git@github.com:pyconjp/bootcamp-text.git
$ cd bootcamp-text
$ virtualenv env
$ . env/bin/activate
(env)$ pip install -r requirements.txt
(env)$ make html
(env)$ open build/html/index.html
```
